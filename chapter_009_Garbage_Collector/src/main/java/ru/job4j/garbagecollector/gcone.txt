Пустой объект в java занимает около 16 байт, которые включают в себя
- ссылку на класс
- информацию о garbage collection
- информацию о синхронизации
String занимает около 40 байт:
- сам объект 16
- по 4 байта на offset, count, hashcode
- ссылку на массив символов char[] - 8
- и вырывнивание 4, т.к. использование памяти выравнивается кратно 8 байтам.
Итого класс с одной строковой переменной будет занимать около 56 байт.
При создании большого количества объектов начинает срабатывать garbage collector,
освобождая место в heap.
